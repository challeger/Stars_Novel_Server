# 群星小说网-后端

**项目目标: ** 实现一个小说网站,用户可以绑定各大主流小说网站的账号,在一个网站上进行小说阅读,订阅,收藏,评论等功能.

#### **拟定字段**: 

**用户**(NovelUser)

**书架**(Shelf): 用户 一对多 书架

#### **拟定接口**:

**用户操作:**

- 用户登录: /user/login/ 发送数据,进行登录,成功返回一个token,需要前端将token加入请求头中.
- 用户注册: /user/register/ 发送数据进行用户注册,注册成功返回注册成功信息..
